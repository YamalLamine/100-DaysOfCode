## üçé macOS Git Setup & Resources

### Initial Configuration
Before using Git on macOS, run these commands to install the necessary tools and link your credentials to the Mac Keychain:

* `xcode-select --install` ‚Äî Installs Apple's Command Line Tools (required for Git).
* `git config --global credential.helper osxkeychain` ‚Äî Tells Git to remember your passwords using your Mac's secure Keychain.

### Official & Mac-Specific Git    Resources
* [Apple Developer: Source Control Management](https://developer.apple.com/documentation/xcode/source-control-management) ‚Äî Official guide on using Git within the Apple ecosystem and Xcode.
* [GitHub: macOS Keychain Guide](https://docs.github.com/en/get-started/git-basics/updating-credentials-from-the-macos-keychain) ‚Äî Detailed instructions for managing Git security and tokens on Mac.
* [Git Official Reference Manual](https://git-scm.com/docs) ‚Äî The complete web-based manual for every Git command.
* [Pro Git Book (macOS Setup Guide)](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git) ‚Äî The "Bible" of Git, featuring a deep dive into macOS configuration.

### Local Documentation
To view the technical manual for your specific version of Git without leaving the terminal, run:
`man git`

> **Navigation Tip:** In the `man` viewer, use **Spacebar** to scroll and press **q** to quit.
